# Phase 1 status

The first REACT Camp desktop vertical slice includes:

- Windows desktop packaging with Tauri
- a camp dashboard and Day 1 interactive lesson
- local Ollama model status and streaming chat
- five-minute idle protection for slow local model starts
- saved chat sessions, prompt history, settings, and lesson progress
- the original full camp curriculum available inside the app

The GitHub Actions build produces an unsigned Windows installer artifact. Code
signing and the remaining camp-day activities are planned for later phases.
