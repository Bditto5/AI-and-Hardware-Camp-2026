# Phase 5 status

Version 0.5 improves classroom presentation and student project portability.

## Presenter mode

- full-screen, distraction-reduced lesson layout
- enlarged titles, body text, cards, and navigation
- Right Arrow or Page Down advances
- Left Arrow or Page Up goes back
- Escape leaves presenter mode
- normal slide position and completion persistence remain active

## Build Lab portability and recovery

- duplicate the current project with all recovery snapshots
- export a standalone HTML page
- export a validated `.react-camp.json` project bundle
- import project bundles as new projects without overwriting existing work
- confirmation before destructive project deletion
- automatic `Before restore` snapshot before restoring older code
- delete individual recovery snapshots
- clear status feedback after import, export, duplicate, restore, or removal
