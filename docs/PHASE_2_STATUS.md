# Phase 2 status

Version 0.2 completes the native five-day learning arc and begins migrating the
interactive practice modules out of the preserved browser curriculum.

## Included

- 42 structured lesson slides across all five days
- selectable day navigation and independent slide resume positions
- saved completion state for every day
- automatic continuation of the next incomplete day
- Hardware Quiz with ten questions and answer explanations
- AI Quiz with ten questions and answer explanations
- five-part Bug Hunt with hints and corrected-code comparisons
- locally saved best quiz and Bug Hunt scores

## Next phase

The next build will migrate Activities and the Build Lab, including named code
projects, autosave, safe preview, snapshots, and the local AI code coach.
