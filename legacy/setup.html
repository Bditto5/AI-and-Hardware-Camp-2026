🚀 Open AI Helper
1

Run start-ollama.bat

Find the file start-ollama.bat in the same folder as this page. Double-click it. A black window will open — that's normal. Wait until it says "Ollama is running!"

2

Keep that window open

Don't close the black window — Ollama needs it to stay open. You can minimize it. If you accidentally close it, just double-click start-ollama.bat again.

3

Check the connection

Once the bat file shows "Ollama is running", come back here and click the button below to check again.

🔌 First time? The bat file will automatically download the AI model (~2 GB). This only happens once — it may take 5–10 minutes on your first run. After that it starts in a few seconds.
